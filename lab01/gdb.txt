1. Для установки параметров запуска используют комманду set args [args]. Если программа уже запущена, то изменения будут видны только после рестарта. Команда show args выводит в консоль установленные аргументы.
2. Точка останова создается командой break [file]:[line]. В случае установки точки в нынешнем файле, можно использовать команду в виде break [line]. Также, можно устанавливать точку останова на функцию: break [functionname]
3. Следующий после точки останова шаг выполняется командой step. Возможно сокращение: s
4. Для того, чтобы выполнить функцию целиком, можно использовать команду next. Возможно сокращение: n. Для того, чтобы попасть внутрь функции, нужно использовать команду step.
5. Для продолжения выполнения программы нужно использовать команду continue. Возмодно сокращение: c
6. Команда print [variablename] выводит в консоль значение переменной. Ввод команды print 1+2 выведет в консоль результат.
7. Команда display [varibalename] выводит значение перемнной после каждой остановки программы. Для отслеживания можно добавлять несколько переменных. Удаление переменной из списка отслеживания выполняeтся командой delete display [varNumber]
8. Команда info locals выводит в консоль значения всех переменных в области видимости.
9. Команда quit заевршает работу дебагера. Возможно сокращение: q
